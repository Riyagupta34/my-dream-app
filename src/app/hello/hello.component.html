Data Binding

Angular Databinding is used for communication. It is used to communicate between your TypeScript code (your business logic) and the other component which is shown to the users i.e. HTML layout.

String Interpolation - is used to display dynamic data on HTML template (at user end). It facilitates you to make changes on component.ts file and fetch data from there to HTML template. The Interpolation markup symbol is use to perform String Interpolation in template(HTML)file.
